Sistema de Controle de Estoque
---------------

API desenvolvida em Python para gerenciamento de estoque de produtos de uma loja de roupas.
O sistema permite cadastrar produtos, consultar estoque, obter quantidades e remover itens do banco de dados.
______________
Tecnologias utilizadas

Python

FastAPI

SQLAlchemy

SQLite

Uvicorn
_________________________________

Funcionalidades

A API permite realizar as seguintes operações:

Cadastrar produtos no estoque

Listar produtos cadastrados

Obter quantidade do produto em estoque

Deletar produtos do estoque
_____________________________
Objetivo do projeto:

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento backend, conforme a minha realizade:

criação de APIs REST

integração com banco de dados

uso de ORM

organização de código
